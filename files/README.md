

Features

- All 36 Maharashtra districts with real agricultural data
- Updated district names: Ahilyanagar (not Ahmednagar), Chhatrapati Sambhajinagar (not Aurangabad), Dharashiv (not Osmanabad)
- Multiple crops per season per district — reflects actual cultivation patterns
- Language switching: English / Hindi / Marathi — all sections update
- All sections update when district changes
- District-specific pest zones (cotton belt, soybean belt, onion belt, konkan rice, sugarcane zone, konkan mango, banana zone)
- Crop calendar specific to selected crop
- No references to any external system in data or UI
